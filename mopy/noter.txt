LightScript version 2.0.0

    Node properties
        - node type
        - data type
        - children nodes
        - position in file
        - comment (line above, lasting until next comment)

    Statement types
        - pass
        - global ...
        - def (def ...)
            - default value
            - docstring
        - class (class Name:)
            - class properties
            - methods
        - for (for ... in ... )
          - limited to lists, strings, tables
        - while 
        - cond (if ... elif ... else ... )
        - try (try ... except ExceptionClass, name ... )
        - raise
        - return
        - assignment (... = ...)
        - subscript-assignment (bhjbh.hjh = ... , nnjnj[123] = ...)
        - (expression)

    Expression types
        - subscript
        - slice[:]
        - comparison: (< > == <= >= in)
        - self/this
        - variable-resolve
        - method call
        - function call
        - arithmetic (+ - * /)
        - literal ("string", True/False, 123, None)

    Builtin functions
        - str(obj)
        - iter(obj)
        - len(obj)
        - Exception(str)
        - file(name)
        - file.read()
        - print(x)
        - dict()
