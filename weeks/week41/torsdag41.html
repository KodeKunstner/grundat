Forms til python

  • Kommunikation via HTTP-protokollen, håndteret via pythons cgi-modul
  • Eksempel:

import cgi, cgitb
cgitb.enable()
print("Content-Type: text/html\n")
fields = cgi.FieldStorage()

for name in fields:
  print(name + ": " + fields[name] + "<br />")
